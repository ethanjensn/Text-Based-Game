# Text-Based Adventure Game

## Overview
This is a simple text-based adventure game written in Python. The objective is for the player to collect all items in the game before encountering the villain. Each step provides descriptive text about the player's current location and prompts the player to decide whether to collect items found in each room.

## Features
- Interactive text-based gameplay
- Multiple rooms to explore
- Item collection system
- Simple win/lose conditions based on game choices
- Includes flowchart and room descriptions for game structure visualization

## Files
- **TextBasedGame.py** – The main Python script containing the game logic.
- **Rooms and Description.pdf** – Details of each room and the items they contain.
- **Text Game Flowchart.pdf** – Flowchart showing game progression and player decisions.

## How to Play
1. Run `TextBasedGame.py` in a Python 3 environment.
2. Read the text prompts carefully.
3. Decide whether to pick up items in each room by typing your response.
4. Collect all items before reaching the villain's room to win.
